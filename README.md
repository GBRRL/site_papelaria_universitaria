# Site Papelaria Universitária

Customização de layout em loja virtual na plataforma Nuvemshop por meio de ajustes em CSS, organização de elementos visuais e melhoria da estrutura da página. Tratamento e otimização de imagens para garantir maior qualidade visual e melhor apresentação dos produtos.

# Descrição

Customização do layout da loja utilizando CSS para ajustes visuais, organização de elementos e melhoria da experiência do usuário.

Também foi realizado tratamento e otimização de imagens para garantir melhor qualidade visual dos produtos.

# Tecnologias utilizadas

- CSS
- Edição de imagens
- Customização de layout
